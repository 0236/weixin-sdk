#微信公众平台(订阅号、服务号、企业号)SDK

weixin-sdk 是对微信公众平台API的封装：

 - 服务号，订阅号：weixin-mp
 - 企业号：weixin-qydev

##如何获取？

 - 服务号，订阅号：

		<dependency>
	      <groupId>cn.com.riversoft</groupId>
	      <artifactId>weixin-mp</artifactId>
	      <version>0.2.0</version>
	    </dependency>

 - 企业号：
 
	    <dependency>
	      <groupId>cn.com.riversoft</groupId>
	      <artifactId>weixin-qydev</artifactId>
	      <version>0.2.0</version>
	    </dependency>
